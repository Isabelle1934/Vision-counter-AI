# Vision-counter-AI

## About the Project

VisionCounter AI is a Computer Vision project developed in Python that performs real-time people detection and counting using Artificial Intelligence.

The system captures images from a webcam and uses the YOLO11 (Ultralytics) model to automatically detect people in the scene, displaying the number of detected individuals in real time.

This project was developed to deepen knowledge in Computer Vision, Deep Learning, and Artificial Intelligence, serving as a foundation for future intelligent monitoring and environmental analysis applications.

## Features

- Real-time people detection.
- Automatic people counting using Artificial Intelligence.
- Real-time display of the detected people count.
- Real-time image processing through a webcam.
- Simple, lightweight, and easy-to-use interface.
